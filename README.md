# Spotify API Lib

A modern **C++ library for the Spotify Web API** providing full access to albums, artists, playlists, player controls, and more. 

Built with easy authentication and easy access to endpoints in mind!

## Features
- Full access to Spotify Web API endpoints:
  - Albums, Tracks, Artists 
  - Playlists & Library 
  - Player control (get playback state, currently playing, devices)
  - Shows, Episodes, Audiobooks 
  - Browse, Search & Categories
- OAuth2 Authorization Code Flow for authenticated requests 
- Easy handling of Spotify IDs and URIs 
- Exception-based error handling 
  - APIException, NetworkException, InvalidIDException, etc...
- Example scripts showing real-world use cases
